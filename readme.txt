== Description ==
IndianPost WooCommerce Integration allows seamless integration of India Post services with WooCommerce.  
Features include:
- Automatic Order Booking  
- Real-time Tracking Updates  
- Tariff Calculation  
- Pincode Validation  
- Custom Shipping Rules (Weight, Pincode, Product Type)  
- WooCommerce Admin Settings Panel  

== Installation ==
1. Upload `indianpost-woocommerce` to the `/wp-content/plugins/` directory.  
2. Activate the plugin via the 'Plugins' menu in WordPress.  
3. Go to WooCommerce → Settings → India Post to configure API keys.

== Frequently Asked Questions ==
= How do I get India Post API credentials? =
You need to register with India Post and obtain API credentials.

= Does it support live tracking? =
Yes, it uses webhooks to update tracking information in WooCommerce.
